NumPy 는 Python 언어의 확장 라이브러리로, 대규모 고차원 배열 및 행렬 연산을 지원합니다. 또한 배열 연산을 위한 다양한 수학 함수 라이브러리를 제공합니다. 머신 러닝은 배열에 대한 많은 변환과 연산을 포함하므로 NumPy 는 필수적인 도구 중 하나입니다.
